rustDFS is a distributed file system written in Rust.
It splits files into blocks and replicates them across data nodes.
A name node tracks which blocks belong to which file, and which
data nodes hold each block. Data nodes store the raw block bytes.

When writing a file, the client streams chunks to the name node,
which assigns a UUID block ID to each chunk, selects a primary
data node and replica nodes, and forwards the write. On read, the
name node returns a stream of block data reassembled in order.

This file is used as the seed for generating medium and large
test files. The content is intentionally ASCII so diffs are
readable if an integrity check fails.
